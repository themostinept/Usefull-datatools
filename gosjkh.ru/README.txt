Это простой парсер списков многоквартирных домов, хранящихся на сайте gosjkh.ru в разделе houses.
В списках отражены адреса домов, названия УК, телефоны УК, год постройки дома, общая площадь и число жителей.
Для корректной работы парсера используется несколько измененная функция html_table (источник кода: https://stackoverflow.com/questions/37293830/specifying-column-class-in-html-tablervest, автор: пользователь hrbrmstr).
Изменения к этой функции позволяют контролировать тип данных в колонках таблиц, которые получаются на выходе функции html_table. Поскольку действует автоматическое приведение типов, в зависимости от таблицы в html, функция html_table может одну и ту же строку сохранять либо как integer, либо character, что делает невозможным объединение двух таблиц в общий датафрейм.
Функция принимает на вход один аргумент: ссылку, ведущую на страницу любого региона со страницы http://gosjkh.ru/houses (пример: http://gosjkh.ru/houses/magadanskaya-oblast). На выходе функция сохраняет результат парсинга как глобальную переменную 'out', а также записывает ее в xlsx файл.
NB: Для записи в xlsx файл используется пакет openxlsx. Для работы функции записи в xlsx файл требуется, чтобы на рабочем компьютере была установлена Java.
